Question-1: What are the different ways to select an element in the DOM?

1. getElementsByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML use when we need HTML tags.
innerText use when we want only visible text.
textContent use when we want all text.

Question-3: What is event delegation in the DOM?

Event delegation means instead of adding events to each button, add one event to the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling means an event starts from the child element and moves up to the parent elements in the DOM

Question-5: How do you create, add, and remove elements using JavaScript?

1.Create an element:
const newDiv = document.createElement('div'); newDiv.textContent = "Hi friend! I'm DOM. "; newDiv.classList.add('my-class');
2.Add and element:
const parent = document.getElementById('container'); parent.appendChild(newDiv);

parent.removeChild(newDiv); newDiv.remove();
